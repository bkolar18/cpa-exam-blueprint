## Quick Reference: Common Gap-to-Topic Mappings

When you see this gap → Practice these topics in Quiz Config:

| Gap Detected | Practice Topics |
|--------------|-----------------|
| **FAR-IV: State and Local Governments** | Government Accounting, Not-for-Profit Accounting |
| **BAR-III: State and Local Governments** | Government Accounting, Not-for-Profit Accounting |
| **FAR-I: Conceptual Framework** | Conceptual Framework & Standards, Financial Statement Presentation, Statement of Cash Flows, IFRS, Fair Value, Accounting Changes |
| **FAR-II: Select Financial Statement Accounts** | Inventory, PP&E, Investments, Liabilities, Revenue Recognition, Income Taxes, etc. |
| **FAR-III: Select Transactions** | Leases, Business Combinations, Consolidations, Derivatives, Foreign Currency |
