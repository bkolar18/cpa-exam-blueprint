# TBS Systematic Polish Plan

## Research Summary: How Citations Appear on the CPA Exam

Based on research from AICPA resources, Becker, and authoritative citation guides:

### Citation Format Standards

**FASB ASC (Accounting Standards Codification):**
- Format: `ASC XXX-YY-ZZ-PP` where:
  - XXX = Topic (e.g., 606 = Revenue)
  - YY = Subtopic (e.g., 10 = Overall)
  - ZZ = Section (e.g., 25 = Recognition)
  - PP = Paragraph (e.g., 1)
- Examples:
  - `ASC 606-10-25-1` (Revenue recognition, paragraph level)
  - `ASC 842-10` (Leases, subtopic level)
  - `ASC 740` (Income Taxes, topic level)
- On exam: Candidates search codification and enter citation in text field

**AU-C (AICPA Clarified Auditing Standards):**
- Format: `AU-C XXX` or `AU-C XXX.PP`
- Examples:
  - `AU-C 315` (Risk Assessment)
  - `AU-C 240` (Fraud Consideration)
  - `AU-C 700` (Audit Reports)
- PCAOB equivalent: `AS XXXX` (e.g., `AS 2401` for fraud)

**IRC (Internal Revenue Code):**
- Format: `IRC §XXX` or `IRC Section XXX`
- Also acceptable: `26 USC §XXX`
- Examples:
  - `IRC §168` (Depreciation/MACRS)
  - `IRC §351` (Corporate Formation)
  - `IRC §721` (Partnership Formation)

**Treasury Regulations:**
- Format: `Reg. §X.XXX-X`
- Example: `Reg. §1.61-2` (interprets IRC §61)

**Circular 230:**
- Format: `Circular 230 §10.XX`
- Example: `Circular 230 §10.27` (Contingent Fees)

## Polish Categories

### 1. Authoritative Citation Enhancement
Add explicit citations to explanations where missing.

**Target pattern for explanations:**
```
"Per [Authority], [explanation of why answer is correct]"
```

**Examples:**
- FAR: "Per ASC 855-10-25-1, conditions existing at the balance sheet date require adjustment (Type I recognized event)."
- AUD: "Per AU-C 240.29, unpredictable audit procedures are required responses to identified fraud risks."
- REG: "Per IRC §168(k), 100% bonus depreciation applies to qualified property placed in service before 2027."
- TCP: "Per IRC §951A, GILTI equals tested income minus net deemed tangible income return."

### 2. Scenario Text Enhancement
Add authoritative context to scenario instructions where candidates would reference standards.

**Target pattern:**
```
"Per [Authority], evaluate/determine/calculate..."
```

### 3. CPA Exam UI Alignment
- Use standard account titles matching CPA exam interface
- Clarify sign conventions for numeric entries
- Add entry structure guidance for journal entries

### 4. Classification Criteria Clarity
- Add explicit criteria for Type I/II subsequent events
- Specify probability thresholds for contingencies
- Label temporary differences as taxable/deductible

## Execution Plan

### Estimated Scope
- ~500 TBS total
- ~60% need citation enhancement (~300 TBS)
- ~20% need UI/format fixes (~100 TBS)
- ~10% need clarity improvements (~50 TBS)

### Approach Options

**Option A: Section-by-Section**
- Complete FAR → AUD → REG → TCP → BAR → ISC
- Easier to track progress
- ~6 work sessions

**Option B: Issue-by-Issue**
- Pass 1: Add all missing citations
- Pass 2: Fix all UI issues
- Pass 3: Add all clarity improvements
- More efficient pattern matching
- ~3 work sessions

**Option C: Prioritized Hybrid**
- Start with sections having most issues (BAR, ISC, TCP)
- Then polish already-strong sections (FAR, AUD, REG)
- Quality-focused approach
- ~4-5 work sessions
